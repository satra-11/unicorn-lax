# Unicorn Lax 要件定義書

## 1. プロジェクト概要

### 1.1 目的・ゴール

- **解決する課題:** アルバム作成時に、大量の写真から様々な条件で写真を選定する作業を効率化する。
- **主なターゲット:** 幼稚園の先生、保育士、学校の先生、親
- **KPI:** 写真の選定作業時間を90%削減する。

### 1.2 コア機能

- 保育園でアルバムを作成する際に、どの子も均等に写真が自動選定されるようにする。
- 保育園でアルバムを作成する際に、特定の子供の成長過程が分かるように、写真を自動選定する。
- 「この写真は〇〇君ですか？」のような質問に回答することで、重心を調整して、分類精度を向上させる。

### 1.3 スコープ

#### 対象範囲

機能範囲:

- 一般ユーザー向けフロントエンド機能

プラットフォーム:

- PC版ブラウザ（Chrome, Edge, Safari 最新版）

業務工程:

- UI/UXデザイン作成
- システム設計・開発・単体/結合テスト
- 本番環境（Cloudflare）へのデプロイ作業

#### 対象外

プラットフォーム:

- iOS/Android ネイティブアプリの開発
- IE11等、レガシーブラウザでの動作保証

データ・運用:

- リリース後のコンテンツ登録・運用代行
- Cloudflareは静的資産（HTML/JS/モデルファイル）の配信のみを担い、画像処理の計算リソースやデータストレージは利用しない。

インフラ費用:

- サーバー利用料、ドメイン維持費の実費

## 4. 非機能要件 (Non-Functional Requirements)

- **パフォーマンス:** 画面描画は1秒以内、APIレスポンスは500ms以内を目指す(ただし、画像解析時を除く)。
- **セキュリティ:** 個人情報を扱うため、個人情報保護法に準拠する。クラウドへのアップロードは行わない。
- **対応OS/ブラウザ:** Chrome, Edge, Safari 最新版
- **可用性:** メンテナンス時以外の稼働率99.9%。
